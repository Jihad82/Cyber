Introduction:

  • Playfair Cipher: Playfair cipher is a polygraphic substitution cipher that was invented by Charles Wheatstone in 1854. It uses a 5x5 grid of letters to encrypt plaintext into ciphertext.
  • Vigenere Cipher: Vigenere cipher is a polyalphabetic substitution cipher that was invented by Giovan Battista Bellaso in the 16th century. It uses a keyword to encrypt plaintext into ciphertext.
  • Multiplicative Cipher: Multiplicative cipher is a monoalphabetic substitution cipher that uses modular arithmetic to encrypt plaintext into ciphertext.
  • Rail Fence Cipher: Rail fence cipher is a transposition cipher that was used by the ancient Greeks and Spartans. It rearranges the plaintext by writing it diagonally over a number of "rails" or lines, then reading off the ciphertext from left to right.

Solution Methodology:

Playfair Cipher:

  1. Generate a key by removing duplicate letters and adding a filler letter if needed.
  2. Create a 5x5 matrix using the key and remaining letters of the alphabet.
  3. Break the plaintext into pairs of letters and apply the following rules:
  4. If the letters are in the same row, replace each letter with the letter to its right (wrap around if needed).
  5. If the letters are in the same column, replace each letter with the letter below it (wrap around if needed).
  6. If the letters are not in the same row or column, create a rectangle with the two letters and replace them with the letters on the opposite corners of the rectangle.
  7. Repeat steps iii to vi for the entire plaintext.

Vigenere Cipher:

  1. Generate a key by repeating a keyword until it is the same length as the plaintext.
  2. Convert each letter of the plaintext and key to their corresponding numerical values.
  3. Add the numerical values of the plaintext and key modulo 26.
  4. Convert the resulting values back to letters.
  5. Repeat steps iii to iv for the entire plaintext.

Multiplicative Cipher:

  1. Choose a multiplier that is coprime to 26 (i.e., does not share any factors with 26).
  2. Convert each letter of the plaintext to its corresponding numerical value.
  3. Multiply the numerical value of each letter by the multiplier modulo 26.
  4. Convert the resulting values back to letters.
  5. Repeat steps iii to iv for the entire plaintext.

Rail Fence Cipher:

  1. Write the plaintext diagonally over a specified number of rows (rails).
  2. Read off the ciphertext from left to right, starting at the top row and moving down to the bottom row in a zigzag pattern.
  3. Repeat steps i to ii for the entire plaintext.
